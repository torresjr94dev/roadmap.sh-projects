
# Single-Page CV

Proyecto oficial: [https://roadmap.sh/projects/single-page-cv](https://roadmap.sh/projects/single-page-cv)

Este proyecto es un Currículum Vitae (CV) de una sola página construido únicamente con HTML semántico, optimizado para SEO y preparado para futuras mejoras de estilo con CSS.

## Características
- Estructura HTML semántica usando `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`.
- Secciones para presentación profesional, habilidades técnicas, experiencia laboral, educación e idiomas.
- Meta tags SEO y Open Graph para mejor visibilidad en buscadores y redes sociales.
- Favicon y enlaces a portafolio profesional.
- Listo para ser estilizado con CSS o frameworks modernos.

## Uso
1. Clona este repositorio.
2. Abre `index.html` en tu navegador para ver el CV.
3. Personaliza el contenido según tus necesidades.
4. (Opcional) Agrega estilos CSS para mejorar la presentación visual.

## Estructura del archivo principal
- `index.html`: Contiene todo el contenido y la estructura del CV.
