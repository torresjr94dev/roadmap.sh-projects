
# Basic HTML Website

> URL del proyecto: [https://roadmap.sh/projects/basic-html-website](https://roadmap.sh/projects/basic-html-website)

Este directorio contiene un sitio web básico hecho únicamente con HTML. Incluye las siguientes páginas:

- `index.html`: Página principal
- `articles.html`: Listado de artículos
- `projects.html`: Proyectos
- `contact.html`: Página de contacto

## ¿Cómo correr el proyecto?

1. **Descarga o clona este repositorio.**
2. Ve a la carpeta `Basic HTML Website`.
3. Abre el archivo `index.html` (o cualquier otro `.html`) en tu navegador web favorito. Puedes hacerlo de varias formas:
	- Doble clic sobre el archivo.
	- O haz clic derecho y selecciona "Abrir con..." y elige tu navegador.
	- O arrastra el archivo al navegador.

No necesitas instalar nada ni ejecutar servidores. Todo funciona localmente.
